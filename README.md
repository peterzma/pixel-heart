# Pixel Heart

## Overview

Pixel Heart is a 2D black-and-white platformer developed in Godot. You navigate through 30 progressively challenging levels, collecting all hearts to progress while using advanced platforming mechanics.

## Features

* 2D black-and-white pixel-art style
* 30 levels with increasing difficulty
* Collect all hearts in a level to progress
* Game timer tracks how long it takes to complete a level
* Retry or proceed to next level after completing a level
* Double jump, wall jump, and variable jump height
* Custom music and sound effects
* Reveal obstacles below by pressing **S** or Down arrow
* Level counter shows your current level
* Simple and responsive keyboard controls

## Getting Started

### Prerequisites

* Godot 3.5 or later
* Windows, macOS, or Linux export templates

### Installation

Clone the repository:

```bash
git clone https://github.com/quackerhacker/pixel-heart.git
```

Open the project:

1. Open Godot
2. Click **Import**
3. Select the cloned repository folder
4. Open the project

Run the game:

1. Click the **Play Scene** button or press **F5**

### Playing the Game

You can also play by downloading the latest release: **[v1.0.0](https://github.com/quackerhacker/pixel-heart/releases/tag/v1.0.0)**

## How to Play

* **Move left/right:** A/D or Left/Right arrow keys
* **Jump:** W or Up arrow key
* **Double jump, wall jump, and hold jump for longer jumps**
* **Reveal obstacles below:** Press **S**
* **Interact with buttons (retry, next level, etc.):** Spacebar or Enter
* **Complete the level by collecting all hearts**
* **Timer** tracks how long it takes
* **Level counter** shows your current level
